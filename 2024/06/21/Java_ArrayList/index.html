集合：ArrayList

Arraylist可以看作功能更多的数组

1.自动扩增

2.自带增删查改

但是注意，数组可以存基本数据类型和引用数据类型，而ArrayList只能存引用数据类型，存基本数据类型时要用对应的包装类。

怎么实现自动扩增？

ArrayList扩容机制：

  1. 利用空参创建的集合，会在底层创建一个默认长度为0的数组

  2. 添加第一个元素时，底层创建一个长度为10的数组

  3. 添加元素发现存满时，扩容1.5倍

  4. 如果一次性添加元素超过一次扩容后的容量，新创建数组的长度以实际为标准

调用集合对象的add方法，尝试向集合中添加一个元素

进入ArrayLIst内部的add方法，这个方法用来判断是否需要扩容

需要扩容，先调用空参方法grow()

再调用含参方法grow(int minCapacity)

当集合不为空时，进入if，调用newLength方法，参数（原数组长度，理论需要添加的最小容量，原数组的0.5倍）

如果只是add方法，那一般扩增1.5倍

如果一次添加了多个元素（如addAll方法），那么需要的空间可能大于扩容1.5倍(oldLength+preGrowth)后的数组。因此进行判断，添加元素后所需空间大于原来的1.5倍则直接扩容为所需大小（oldLength+minGrowth）。
