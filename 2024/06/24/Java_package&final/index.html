包&final修饰

包

作用

是文件夹，用来管理不同功能的java类

书写规则

全英文小写

公司域名反写+包的作用

全类名

包名+类名

什么时候需要导包

使用同一个包中的类时，不需要导包

使用java.lang包中的类时，不需要导包

其他情况需要导包

使用不同包中的同名类时，需使用全类名

final修饰

类

该类为最终类，无法被继承

方法

此方法为最终方法，不能被重写

变量

是个常量，只能赋初值

什么是常量

”见名知意”

实际开发中，一般做为系统的配置信息，方便维护，提高可读性。

一个常量可以命名为它的值所代表的意思，这个名字的意思使得代码具有更高的可读性

命名规范

单个单词：全部大写

多个单词：全部大写，单词间用下划线隔开

此外，变量中两种不同的数据类型在被final修饰时有不一样的表现：

基本数据类型：记录的值不能发生改变

引用数据类型：记录的地址值不能发生改变，但内部的属性值可以改变

归根结底都是常量的定义“是个常量，只能赋初值”

int a = 10;初值是个数

Animal animal = new animal();初值是个地址

关于String的理解

String类型为什么不能被修改？——final修饰

但String不是引用类型吗，为什么不能修改值？ ——private修饰

private final byte[] value;

String底层是一个字节数组，因此value记录的地址值不能改变，而value的地址由于private修饰，外部无法获取其地址，又因为String没有提供获取其地址的方法（get())，所以无法定位到这个数组进行值的修改。
