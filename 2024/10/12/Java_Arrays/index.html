Arrays

操作数组的工具类

常用方法

  • public static String toString(数组)
    把数组拼接成一个字符串

  • public static int binarySearch(数组，查找的元素)
    二分查找法查找元素

  • public static int[] copyOf(原数组新数组长度)
    拷贝数组

  • public static int[] copyOfRange(原数组,起始索引,结束索引)
    拷贝数组(指定范围)

  • public static void fill(数组,元素)
    填充数组

  • public static void sort(数组)
    按照默认方式进行数组排序

  • public static void sort(数组，排序规则)
    按照指定的规则排序

toString

底层就是拼接(当前元素.toString()+’,’)，判断当前是否为最后一个元素，是则拼接’]’

binarySearch

不给区间时，内部会自动添加两个参数(0，数组长)，并调用方法

binarySearch0(a, 0, a.length, key);

进入函数启动二分查找，找不到时会返回这个值本来应该会在的位置+1的相反数

无符号右移：>>>

copyOf

底层调用arraycopy

arraycopy

IntrinsicCandidate注解：与Java的即时编译器（JIT，Just-In-Time Compiler）优化有关，JIT编译器在运行时可以识别这些标记的代码，并尝试将它们转换为更高效的机器码，以提高程序的执行效率。

native关键字：原生方法，指用Java以外的语言（如C或C++）编写的方法，这些方法需要通过Java Native Interface（JNI）在Java代码中调用。原生方法允许Java程序执行一些Java语言本身不支持的操作，比如直接访问硬件。

copyOfRange

底层同上，只是添加了一个范围（取左不取右）

fill

向目标数组中填充指定值

可以接收多个参数，表示填充的区间

rangeCheck中会判断区间的合法性

  • 起始索引是否大于0

  • 终止索引是否大于起始索引

  • 终止索引是否小于数组总长

sort

底层是插入排序+二分查找

插入排序会将数组分为有序段与无序段，每次从无序段中取一个元素放入有序段，由于有序段是有序的，可以使用二分查找：新元素与有序段中间的元素比较

sort默认升序，但我们可以提供自己的比较规则Comparator，由于这是一个接口，需传入其实现类：

使用匿名内部类，o1是无序段取出的那个元素，o2是有序段元素，排序结果通常有三类：

比较结果<0：新元素继续与前面的数据段比较

比较结果>0：新元素继续与后面的数据段比较

比较结果=0：新元素继续同样与后面的数据比较

  • o1-o2即新元素-当前位元素，>0：新元素较大，向后排 》》大的向后：升序

  • o2-o1即当前位元素-新元素，>0：新元素较小，向后排 》》小的向后：降序

  • 等于0表示当前位元素=新元素，根据插入排序的处理方式，新元素应放在其后面，继续比较的原因在于防止后一个元素还是相同元素
