Router前端路由

url中的hash与组件之间的对应关系

Vue的路由由三部分组成：

VueRouter

路由器类，根据路由请求在路由视图中动态渲染选中的组件

等于一个库，里面包含了很多组预先设置好的组件组合

router-link

请求连接组件，浏览器会解析成超链接

向VueRouter查询某个组件组合

router-view

动态视图组件，用来渲染展示与路由器路径对应的组件

可以在任意区域渲染库中的组件组合，不用整个页面刷新而是以块的形式刷新。

使用方式

安装

npm install vue-router

定义路由

  • path：路由路径

  • name：路由名，用于在代码中引用这个路由

  • component: () => import(）定义了当路由被激活时应该渲染的组件

    这里使用了动态import()语法来按需加载EmpView.vue组件。这样做可以提高应用的加载性能，因为只有在用户访问这个路由时，EmpView.vue组件才会被加载，不用再在开头import了！

  • redirect：重定向到另一个路由

此时添加了三个路由，在地址栏输入可以改变页面组件
