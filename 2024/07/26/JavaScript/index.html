Ⅰ.什么是JavaScript?

web标准中提出网页由三部分组成

  1. HTML：负责网页的基本结构（页面元素和内容）
  2. CSS：负责页面的表现效果（外观、位置等页面样式）
  3. JavaScript：负责页面的行为（交互效果）

类比一下就是：

MC中

*这里有一个东西

HTML:“这是个物品”

CSS:”这是个牛扒“

JavaScript:”可以吃耶，真不错”

*这里又有一个东西

HTML:“这是个生物”

CSS:”这是个creeper，长得真别致“

JavaScript:”*你被炸死了——他会爆炸”

这个时候你知道了，JavaScript,也就是本专题的主角，是网页的灵魂，它能为网页提供很多有趣的行为，让网页”活“起来。

特点

  • 跨平台
  • 面向对象
  • 脚本语言：不需要编译，经过浏览器解释即可直接运行，简化开发过程

Ⅱ.引入方式

1.内部脚本

  • js代码必须位于script标签之内
  • 在HTML文档中，可以在任意地方，放置任意数量的script标签
  • 一般会把脚本置于body标签元素的底部，这样可以改善显示速度

Ⅲ.基础语法

1.书写语法

语法规则

  • 区分大小写
  • 结尾分号可有可无
  • 注释（行// ；块/**/）
  • 大括号”{}”表示代码块

输出语句

  • window.alert()/alert()

    写入警告框——浏览器弹出警告窗

  • document.write()

    写入HTML输出——写入HTML,在浏览器页面中显示

  • console.log()

    写入浏览器控制台——写入浏览器控制台

2.变量

1.js是弱类型语言——变量可以存放不同类型的值

2.js中使用var关键字声明变量

[!NOTE]

var即variable:变量

声明的均为全局变量

变量可以重定义

[!IMPORTANT]

ECMAScript 6 中：

新增let关键字,定义局部变量（仅let所在代码块内有效），且不允许重复声明

新增const关键字，声明一个只读常量，不可改变

3.变量名需要遵循以下规则：

  • 不以数字开头
  • 任意字符组成
  • 建议用驼峰法命名

[!NOTE]

驼峰命名法：组成名称的多个单词除第一个首字母小写，其余的首字母大写

3.数据类型、运算符、流程控制语句

数据类型

js中有两种数据类型：原始类型和引用类型

原始类型
  • number：数字
  • string：字符串
  • boolean：布尔
  • null：对象为空
  • undefined：当声明的变量未初始化时，该变量的默认值是undefined

[!TIP]

使用typeof运算符可以获取数据类型

运算符

js中的运算符与java

完全一样！

唯一差别：js中有===,名为全等运算符

==在比较时，会进行类型转换，即转换成同类型数据后才进行比较

===在比较时，不会进行类型转换，类型不同会直接返回false

ep:

类型转换

字符串>>数字：

将字符串字面值转为数字，如果字面值不是数字，则转为NaN

[!TIP]

使用了全局方法parseInt()

从第一个字符开始匹配直到最后一个是数字的字符

返回这个数字串的值

否则返回NaN(Not a Number)

其他类型>>boolean

  • number 0&NaN:false /其他:true
  • string 空字符:false /其他:true
  • null&undefined 直接转为false
