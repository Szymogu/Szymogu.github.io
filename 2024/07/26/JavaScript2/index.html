BOM对象

1.基础对象

Array——数组

js中Array对象用于定义数组

  • 定义

​ var name = new Array(元素列表)

​ var name = [元素列表]

  • 访问

​ arr[索引] = 值

[!TIP]

js中的数组相当于java中的集合，数组长度可变，可以存储多种类型数据

  • 属性

    length

    设置或返回数组中元素的数量

  • 方法

    forEach()

    遍历数组中的每个有值的元素，遍历的每次都调用一次传入的函数

    方法一：

    方法二：新版引入了箭头函数，用于简化函数

    push()

    将新元素添加到数组的末尾，并返回新的长度

    *可以一次性添加多个元素

    arr.push(5,6,7);

    splice()

    从数组中删除元素

    参数为：(起始位置，个数)

    `arr.splice(1,2)//从索引1开始删除2个元素

String——字符串

String用于字符串对象创建

    去除字符串两边的空格

    截取指定索引间字符串，含头不含尾（字符串首，尾+1）

3.DOM(文档模型)对象

概念：Document Object Model，文档对象模型

将标记语言的各个组成部分封装为对应的对象

DOM将网页转化为一个由节点组成的树结构，每个节点可以是元素、属性、文本等

有什么用呢？

我们可以使用Js访问DOM树中的节点，修改它们的内容、样式、或者响应用户的事件。即：

DOM提供了一套操作HTML&XML的接口，可以通过脚本访问与操作网页元素。这使得动态网页成为可能。

DOM是w3c的标准，定义了访问HTML和XML文档的标准，分为三个不同的部分

1.Core DOM——所有文档类型的标准模型

  • Document：整个文档对象
  • Element：元素对象
  • Attribute：属性对象
  • Text：文本对象
  • Comment：注释对象

*下面两种均为Core DOM的子集

2.XML DOM——XML文档的标准模型

3.HTML DOM——HTML文档的标准模型

HTML DOM将所有标签都封装成了DOM对象,如

  • Image:<*img*>
  • Button:<*input type=’button’*>

通过DOM的介绍我们知道，DOM是用于操作页面元素的，那整个流程是怎么样的？其实只有两步

1.元素选择

2.操作元素

首先从window对象中获取Document对象，再通过Document对象获取Element对象。

Document对象中提供了以下获取Element对象的函数：

1.根据id属性值获取，返回单个Element对象

document.getElementById('')

2.根据标签名获取，返回Element对象数组

document.getElementsByTagName('')

3.根据name属性值获取，返回Element对象数组

document.getElementsByName('')

4.根据class属性值获取，返回Element对象数组

document.getElementsByClassName('')

例：使用DOM找到id为a的对象，并修改了它的值

元素标签拥有的属性有两个来源，一是标准属性（可用于所有HTML元素），二是元素特有属性
元素属性/方法
HTML DOM Element 对象 (w3school.com.cn)
通过查阅DOM对象的方法，可以操作不同的元素
学习案例中使用了三个属性
img-src：修改图片源
check-checked：复选框状态

BOM&DOM？

BOM：与浏览器（窗口）的交互
DOM：与页面的交互
