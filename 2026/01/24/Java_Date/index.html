Java_时间类

Created2026-01-24|Updated2026-01-24|Java
|Post Views:

时间类

Date类

空参构造会获取本机当前时间，可接受参数类型long，代表当前时间距时间起始位置的毫秒值

setTime方法：修改时间对象的毫秒值

getTime方法：获取时间对象的毫秒值

例：时间开始一年后的日期

SimpleDateFormat

通过模式字符串来定义日期时间的格式

构造函数空参时会创建一个默认的日期格式

可以接收一个String类型参数指定格式

格式化的时间形式的常用的模式对应关系如下：

  • y: 年
  • M: 月
  • d: 日
  • H: 24小时制的小时
  • h: 12小时制的小时
  • m: 分钟
  • s: 秒
  • S: 毫秒
  • E: 星期几
  • D: 一年中的第几天
  • F: 月份中的第几个星期几
  • w: 一年中的第几个星期
  • W: 月份中的第几个星期
  • k: 24小时制的小时（1开始）
  • K: 12小时制的小时（0开始）
  • z: 时区
  • Z: 时区的RFC 822格式
  • v: 时区的通用协调（协调世界时）
  • a: 上午或下午的标记

常用方法

format：日期对象（参数类型Date）-》字符串

parse：字符串（参数类型String）-》日期对象

练习：判断某个时间是否在一段时间内

Canlendar

代表了系统当前时间的日历对象，可以单独修改、获取时间中的年，月，日

Calendar是一个抽象类，不能直接创建对象

使用getInstance()获取当前时间的日历对象

常用方法

public final Date getTime()

获取日期对象

public final setTime(Date date)

给日历设置日期对象

public long getTimeInMillis()
拿到时间毫秒值
public void setTimeInMillis(long millis)
给日历设置时间毫秒值
public int get(int field)

取日历中的某个字段信息

public void set(int field,int value)

修改日历的某个字段信息

public void add(int field,int amount)
为某个字段增加/减少指定的值

注意：

日历类中月份的范围:0~11
日历类中星期的特点:星期日是一周中的第一天

JDK8时间

Date

1.ZoneId:时区

static set getAvailablezonelds()

获取Java中支持的所有时区

static ZoneId systemDefault()
获取系统默认时区

static ZoneId of(string zoneId)
获取一个指定时区

2.Instant:时间戳

static Intant now() 获取当前时间的Instant对象

static Instant ofXxxx(long epochMilli) 初始化一个Instant对象

ZonedDateTime atZone(ZoneId zone) 指定时区

boolean isXxx(Instant otherInstant) 判断

Instant minusXxx(long millisToSubtract) 减

Instant plusXxx(long millisToSubtract) 加

jdk8之后新增的时间类都是不可变的，上面的加减方法都是返回一个新对象，不会改动

3.ZoneDateTime:带时区的时间

static ZonedDateTime now() 获取当前时间的ZonedDateTime对象（带时区）

static ZonedDateTime ofXxx() 获取指定时间的ZonedDateTime对象（带时区）

ZonedDateTime withXxx() 修改时间

ZonedDateTime minusXxx() 减

ZonedDateTime plusXxx() 加

日期格式化类

SimpleDateFormat

Static DateTimeFormatter ofPattern() 获取格式对象（指定格式）

String format() 格式化

日历

LocalDate：年、月、日

LocalTime：时、分、秒

LocalDateTime：年、月、日、时、分、秒

static now() 获取当前时间的对象

static of() 获取指定时间的对象

getXxx() 获取日历中年、月、日、时、分、秒等信息

isBefore() isAfter() 比较

withXxx() 修改（为）

minusXxx() 减

plusXxx() 加

工具

Duration 时间间隔（秒、纳秒）

Period 时间间隔（年、月、日）

ChronoUnit 时间间隔（通用）

between() 计算时间间隔
