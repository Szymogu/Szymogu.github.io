集合进阶（下）

Map

双列集合顶层接口

一次存一对数据：键：值，键不能重复，键值一一对应

键值对对象在Java中称为Entry

常用方法

V put(k key,V value)：添加/覆盖元素

添加时返回null，覆盖时返回被覆盖元素的值

V remove(Object key)： 根据键删除键值对

void clear()：清空容器

boolean containsKey(Object key)：判断容器是否包含键

boolean containsValue(Object value)：判断容器是否包含值

boolean isEmpty()：判断集合是否为空

int size()：集合长度

Map的遍历

keySet：获取所有key

entrySet：获取所有entry

foreach：底层使用 for(:) ，遍历出每个对象调用传入的实现类BiConsumer对象的action方法

foreach源码

HashMap

  • HashMap底层是哈希表结构的
  • 仅使用键区分对象，依赖hashCode方法和equals方法保证键的唯一，因此如果键要存储的是自定义对象，需要重写hashCode和equals方法

LinkedHashMap

  • 多一个双向链表，单个节点会多两个地址，用于记录前后节点位置，因此该容器可记录节点存入顺序

TreeMap

  • TreeMap底层是红黑树结构
  • 依赖自然排序或者比较器排序,对键进行排序
  • 如果键存储的是自定义对象,需要实现Comparable接口或者在创建TreeMap对象时候给出比较器排序规则（底层使用多态将对象转为Comparable，不实现会报错）

三种双列集合，如何选择

默认HashMap（效率最高）

需存取有序：LinkedHashMap

需排序：TreeMap

不可变集合

定义完后不可修改

使用of方法创建不可变集合

细节

初始化Set时元素不可重复

初始化Map时元素不可重复，且上限为10个键值对

超过10个用ofEntries方法（java10以上copyOf）
